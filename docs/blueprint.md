# **App Name**: Siraj Platform

## Core Features:

- User Authentication & Profiles: Allow users to register, log in, and manage their personal profiles, course progress, and completed certifications.
- Course Content & Playback: Display educational courses, manage units, videos, and associated materials. Provide an integrated video player with progress tracking and access control for paid content.
- Digital Certificates & Verification: Generate and manage digital certificates upon course completion, featuring unique IDs and QR codes, with a public verification tool.
- AI Learning Assistant: A generative AI tool to assist users with course navigation, payment inquiries, specialized advice, and frequently asked questions, resembling a smart chat interface.
- Manual Payment & Access Control: Facilitate manual bank transfers for course and book purchases, with an admin panel for subscription activation based on payment verification.
- Basic Admin Content Management: A backend interface for administrators to manage users, courses, activate subscriptions, and oversee core content and verification processes.

## Style Guidelines:

- Primary color: #6D28D9 (A professional, modern purple signifying creativity, technology, and excellence, providing a bold focal point for the interface.)
- Accent color: #10B981 (A vibrant green, used for successful states, confirmations, and elements emphasizing achievement and progress.)
- Secondary contrast color: #2563EB (A technical blue providing trust and professionalism, used for secondary interactive elements and branding accents.)
- Background: #F8FAFC (A clean, very light off-white, serving as a primary background color for visual comfort and a modern aesthetic, complementing the darker text.)
- Text and deep elements: #0F172A (A dark, rich hue ensuring excellent readability for text and providing depth for structural elements like footers.)
- Headline font: 'Space Grotesk' (a sans-serif with a modern, tech-inspired feel, for headings and titles).
- Body text font: 'Inter' (a versatile sans-serif for clear, readable body content and detailed descriptions, complementing the technical aesthetic).
- Utilize a minimalist and professional icon set, consistent with the tech-inspired, modern design language, ensuring clarity and recognition.
- A mobile-first, fully responsive grid-based layout emphasizing clear hierarchy, clean spacing, and a minimal aesthetic, adapting seamlessly across devices.
- Incorporate modern cards with subtle Glassmorphism and professional shadows to group content, enhance visual depth, and maintain a premium feel.
- Subtle, non-distracting hover effects and light micro-animations for interactive elements, enhancing user feedback and interface responsiveness without impeding speed.
